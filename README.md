## 系统介绍


“人效评估管理系统”是一款专注于企业人力效能评估和优化的智能化管理平台，涵盖模型因子管理、指标设置、评分模型和数据来源管理等功能模块。系统支持人效模型、看板模型和体检表模型的编辑和预览，可实现多维度的数据分析和可视化呈现，帮助管理者快速了解人效情况并发现优化空间。同时，系统提供得分段位设置和人效分析模型管理功能，便于对员工效能进行精确评估和分层分析，为企业的人才管理和决策提供科学依据。
本项目名称为人效评估管理系统，旨在对企业的人效进行全面评估与分析。该系统适用于各类企业，主要包括用户管理、角色管理、菜单管理、部门管理、岗位管理、通知公告、日志管理、数据管理、图表维度、图例管理等模块。系统能够帮助企业监控在线用户、定时任务，并提供多维度的人效分析与对比功能，提升企业管理效率。
